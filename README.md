# Simple Object Tracking - OpenCv (color based)
Projeto inicial de visão computacional de um seguidor de objetos com base em cores.
![gifImplementacao](img\WhatsApp-Video-2020-09-28-at-12.55.gif )
Como a trackbar é possivel selecionar o espectro da mascara de captura, com isso é possivel isolar o objeto em questão. Esse tipo de técnica de permite seguir objetos em movimento com facilidade.
